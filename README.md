## libtac
A go lang library for creating text based n dimensional tic-tac-toe games

```
Two
		X|O|
		-|-|
Three
		O|-|-|
		-|-|X|
		-|-|-|
Four
  	        X|-|O|O|
		-|-|O|-|
		-|O|-|-|
		X|-|X|X|
           .
           .
           .
```
